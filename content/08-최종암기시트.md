## ② 암호 — 기술형

| 안전 ✅ | 금지 ❌ |
|---|---|
| 대칭키 **SEED·ARIA·AES·HIGHT·LEA** | DES · 3DES(폐지) · RC4 |
| 해시 **SHA-256/384/512** | SHA-1 · MD5 · MD4 |
| 공개키 RSA(2048↑)·ECC | RSA-1024↓ / CRC32(무결성용·암호X) |

## ③ 의무·시험 기준

- **ISMS 의무**(택1): ISP / IDC / 매출 **100억↑** or 이용자 **100만↑** / 병원·대학 **1,500억↑**(또는 재학생 1만↑)
- **ISMS-P = 자율**(개인정보 포함, 의무 아님) · ISMS 80개 / ISMS-P 101개
- 시험 **50문항·120분·5지선다** / 2026~ **상대평가(60점↑ 중 상위 100명)**
- CISO 지정요건: 자산 5조↑ 또는 ISMS 의무대상 중 자산 5천억↑ → 이사급(겸직 제한)

## ④ 헷갈림 — 시험 최빈출 함정 ⭐

**제3자제공(3.3.1) vs 위탁(3.3.2)** — "누구의 이익이냐"
- 받는 쪽 목적 = **제공**(별도 동의) / 우리 대신 = **위탁**(처리방침 공개·교육·감독)
- 택배·콜센터·**광고 발송 대행** = 위탁 / 제휴사 자기영업용 = 제공

**나머지 1:1**
- 2.5(인증·권한: 계정·식별·인증·비번·권한) ↔ 2.6(접근통제: 네트워크·시스템·응용·DB·무선·원격·인터넷)
- 2.9.4 로그 **관리(생성·보관)** ↔ 2.9.5 로그 **점검(분석)**
- 2.10.8 **패치 미적용** ↔ 2.11.2 **취약점 점검 후 미조치**
- 2.8.3 **환경 미분리** ↔ 2.8.4 **운영 실데이터를 시험에 사용**
- 1.2.3 위험평가(구축) ↔ 1.4.2 관리체계 점검(운영 내부감사)
- 2.2.5 퇴직(통보·회수 부재) ↔ 2.5.1 계정관리 ↔ 2.5.6 권한 재검토 미수행
- 목적외이용 3.2.4(내부 다른 용도) ↔ 제3자제공 3.3.1(외부로)
- 가명정보 = **개인정보(법 적용)** / 익명정보 = 법 적용 X
- 주민번호 = **법령 근거 필수**(동의로도 불가) / 고유식별·민감 = 별도 동의 가능

## ⑥ 시험장 1분 전 주문
1. 결함문제 = **사건의 트리거(발단)**를 짚어 가장 직접적 기준 선택
2. "가장 적절한/핵심" 발문 = 중첩 기준 중 **근본 원인**
3. 2회 풀기 — 확실한 것 먼저, 헷갈리면 표시 후 패스
4. 함정 보기: 묵시동의 / 주민번호 동의OK / 선택동의 거부 시 서비스거부 → **전부 오답**
